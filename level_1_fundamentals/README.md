# Level 1: GenAI Fundamentals
**No frameworks - Pure API understanding**

## Overview
Learn to work directly with LLM APIs without abstraction layers. This builds foundational understanding of how LLMs actually work.

## Projects

### 01_basic_chat - Simple Conversational AI
- Single-turn and multi-turn conversations
- Understanding context and message history
- Temperature, max tokens, and other parameters
- **Healthcare Example**: Patient symptom intake chatbot

### 02_embeddings - Semantic Search
- Generate embeddings for text
- Calculate similarity between texts
- Build simple search without vector DB
- **Healthcare Example**: Find similar medical cases

### 03_function_calling - Tool Use
- Define functions LLMs can call
- Handle function call responses
- Multi-step function orchestration
- **Healthcare Example**: Look up drug interactions

### 04_prompt_engineering - Effective Prompts
- Zero-shot, few-shot, and chain-of-thought
- System prompts and roles
- Prompt templates and variables
- **Healthcare Example**: Clinical note structuring

### 05_streaming - Real-time Responses
- Stream tokens as they're generated
- Better UX for long responses
- Handle streaming errors
- **Healthcare Example**: Real-time medical Q&A

## Learning Objectives

After completing Level 1, you will:
- ✅ Understand tokens, context windows, and pricing
- ✅ Make effective API calls with proper error handling
- ✅ Use embeddings for semantic similarity
- ✅ Implement function calling (tool use)
- ✅ Write effective prompts using proven patterns
- ✅ Stream responses for better UX

## Time Estimate
5-7 hours total (1-1.5 hours per project)

## Next Steps
After mastering these basics, you'll be ready for Level 2 (RAG Systems) where you'll learn to augment LLMs with external knowledge.
