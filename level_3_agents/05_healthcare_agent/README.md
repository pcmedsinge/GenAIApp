# Project 5: Healthcare Agent Capstone — Clinical Decision Support

## What You'll Learn
- Combine RAG + Agent Tools + Multi-Agent workflows
- Safety guardrails for healthcare AI
- Structured clinical outputs (SOAP notes, assessments)
- Production patterns for medical AI systems

## This Project Combines EVERYTHING

| Skill | From Level/Project |
|-------|-------------------|
| LLM API calls, function calling | Level 1 |
| RAG (knowledge retrieval) | Level 2 |
| ReAct agent loop | Level 3, Project 01 |
| LangChain tools | Level 3, Project 02 |
| LangGraph workflows | Level 3, Project 03 |
| Multi-agent collaboration | Level 3, Project 04 |

## Healthcare Applications
- Clinical decision support during patient encounters
- Medical knowledge Q&A for healthcare workers
- Automated clinical documentation assistance
- Medication review and safety checking

## Exercises
1. Add a RAG-backed knowledge base as a tool (combine Level 2 + Level 3)
2. Implement a "human-in-the-loop" pattern for high-risk recommendations
3. Build an audit trail that logs every agent action for compliance
4. Create a patient education agent that explains diagnoses in simple language
